# 开发指南总览

欢迎来到 Yumeri 开发者文档。Yumeri 是一个基于 Node.js 的新一代模块化 Web 应用构建平台，专为极致的开发体验和模块化解耦而设计。

## 快速上手

如果你是第一次使用 Yumeri，请按照以下步骤开始：

1.  **[环境搭建](./setup)**：安装框架并初始化你的第一个项目。
2.  **[插件基础](./plugin)**：了解如何编写你的第一个插件（支持函数式与装饰器 API）。
3.  **[配置构型](./config)**：学习如何定义插件配置并实现类型安全。

## 核心理念

-   **模块化 (Modular)**：一切皆插件，功能高度解耦。
-   **双 API 支持**：自由选择经典的函数式 API 或现代的装饰器 API。
-   **类型安全**：深度集成 TypeScript，提供完美的 IDE 补全。
-   **灵活路由**：基于分段匹配的强大路由系统。

## 进阶主题

-   **[装饰器 API](./decorator)**：深入了解声明式开发。
-   **[中间件](./middleware)**：掌握请求生命周期控制。
-   **[钩子系统](./hook)**：实现插件间的 UI 与功能注入。
