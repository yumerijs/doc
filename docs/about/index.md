# 关于 Yumeri

Yumeri 是一个基于 Node.js 的新一代模块化 Web 应用构建平台。它以“插件 + 中间件”双驱动为核心思想，将应用能力解耦为可插拔的模块，让你在保持灵活性的同时，仍然拥有清晰可维护的架构。

## 设计理念

- **模块化**：核心只提供必要的基础能力，功能以插件形式注入，按需组合。
- **可扩展**：插件与中间件协作，既可扩展业务能力，也可统一处理横切逻辑（鉴权、统计、日志等）。
- **开发友好**：TypeScript 原生支持，类型提示清晰，文档与示例配套。

## 组成结构

- **核心模块 (core)**：框架运行时、基础 API 与生命周期。
- **加载器 (loader)**：负责插件的发现、加载与管理。
- **插件系统**：所有功能模块化封装为插件，形成应用能力的“积木”。
- **中间件系统**：对请求/响应过程进行链式处理。

## 适用场景

- 需要快速搭建、可演进的 Web 应用或管理后台。
- 希望通过插件生态复用能力，减少重复开发。
- 需要按需裁剪、灵活部署的服务端应用。
